# Nylas Email Skill

Access Gmail and other email providers through Nylas API.

## Usage

```
@nylas check email
@nylas send email to john@example.com subject "Hello" body "Message"
@nylas search emails from boss@company.com
@nylas get unread count
```

## Configuration

API key stored in `config.yaml`.

## Tools

- `check_email` - Read latest emails
- `send_email` - Send new email
- `search_emails` - Search by criteria
- `get_unread` - Get unread count
